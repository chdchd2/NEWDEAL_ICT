국방 아키텍처 관리체계란 무엇인가요?
설명 부탁드립니다.
  • 답변
  • 국방 아키텍처 관리체계란 여러 관리체계에서 올라온 데이터를 검색하여 필요한 정보를 찾을 수 있고
    해당 정보의 위치를 알려주어 상세한 정보를 찾아 볼 수 있게 해주는 시스템이며 , 타 체계 인터페이스를 소개하여
    해당 인터페이스와 연계할 수 있도록 해 주는 시스템 입니다.
